## 프로젝트 경험

### 1. AI 여행 일정 추천 서비스 - 도쿄 여행 플래너 (캡스톤 프로젝트)
- 기간: 2025.09.23 ～ 2025.12.11
- 역할: 백엔드 및 데이터베이스 개발, 배포 환경 구성

**프로젝트 소개**  
일본 도쿄 여행을 계획하는 사용자가
간단한 선호도와 여행 일정을 입력하면, AI가
일정에 맞는 관광지·동선·시간 계획을 자동으로 추천해 주는 웹 서비스입니다.

**주요 기여**
- Cloudflare Pages를 사용해 서비스 빌드 및 배포 파이프라인 구성
- Cloudflare D1 Database를 활용해 사용자/여행 일정/장소 데이터를 설계 및 연동
- 회원가입·로그인 기능 구현 및 세션/인증 로직 처리
- 일정 데이터 CRUD API 설계 및 백엔드 로직 구현

**문제 해결 사례**
- 문제: 사용자가 여행 일정을 수정했을 때, 기존에 추천된 교통수단이 새로운 일정에 맞게 자동으로 바뀌지 않는 문제가 있었습니다. 
- 해결: 일정 수정이 완료되면, 해당 일정에 최적화된 교통수단을 자동으로 다시 추천하도록 구현했습니다.
